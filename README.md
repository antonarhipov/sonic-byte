# SYNTHETIC APOCALYPSE
## A Sonic Byte

**8 tracks of Dark Clubbing / Industrial Electronic music composed in Sonic Pi**

---

### About

*Synthetic Apocalypse* is an 8-track album (a "Sonic Byte") created entirely in [Sonic Pi](https://sonic-pi.net/), a live coding music synthesizer. The album explores dark clubbing, industrial electronic, and darksynth territories, drawing inspiration from artists like **Irving Force**, **Noisecream**, **Gesaffelstein**, **1788-L**, and **REZZ**.

Each track is a standalone `.rb` file that can be run directly in Sonic Pi.

### How to Play

1. Download and install [Sonic Pi](https://sonic-pi.net/)
2. Open any `.rb` file from this repository
3. Copy the code into a Sonic Pi buffer
4. Press **Run** (or `Alt+R` / `Cmd+R`)

Each track is self-contained and requires no external samples or dependencies.

---

### Musical Philosophy

The album follows Dark Clubbing production principles:

- **Heavy, punishing bass** - Clean sub frequencies with gritty midrange
- **Industrial textures** - Metallic percussion, noise sweeps, dark atmospheres
- **DJ-functional structure** - Clean intros/outros, 16/32-bar phrases
- **Tension and release** - Builds, breakdowns, and impactful drops
- **Cyberpunk aesthetics** - Dark, futuristic, aggressive

---

### Technical Notes

- All tracks are optimized for Sonic Pi's buffer capacity (~3-6KB each)
- Uses built-in Sonic Pi synths: `tb303`, `prophet`, `dsaw`, `pulse`, `dark_ambience`, `hollow`
- Uses built-in samples: `bd_tek`, `bd_zum`, `bd_haus`, `sn_dub`, `drum_snare_hard`, `drum_cymbal_closed`
- Effects used: `reverb`, `echo`, `lpf`, `hpf`
